January - September, 2020

Hardware Engineer Coop
Sierra Wireless

Worked in the design team, assisting with design tasks, prototyping and PCB assembly and testing.
Performed automotive and ESD testing on several revisions of a design, determining failure points and suggesting and testing possible solutions to ensure compliance to IPC standards.

September - December, 2019

Product Design Engineer Coop
Microchip Technology

Worked in the design team, writing and testing RTL using SystemVerilog for a VLSI device.
Used coverage driven verification techniques to perform unit testing and ensured static timing analysis (STA) and timing closure to ensure acceptable performance.
Wrote clean, readable and maintable code with an emphasis on clear and concise commenting and documentation.
Gained experience in VLSI design using SystemVerilog, SVN version control in a Linux environment and high speed PCI-E controllers.

May - Aug, 2019

Radio Frequency Engineer Coop
Orcasat/Canadian Space Agency

Worked as a member of the RF team, designing and building a communication system for a 3U cubesat.
Used KiCAD to design a space segment RF transceiver using Time Division Duplexing (TDD) with a custom designed ultrawideband antenna. Also designed a low power transmitter capable of delivering a linear chirp to allow calibration of the CHIME radio telescope. Modified code from the OpenLST project to provide a simple interface to the main satellite processor.
Planned and implemented a ground station using Ettus USRPs and GNU Radio.
Gained experience in RF PCB design and testing using Spectrum Analyzers and Vector Network Analyzers, antenna matching and filtering. Also practiced programming in C and python.
